Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How do you select an element from the DOM?

Element can be selected using many selectors. document.getElementByID('id') is used to find element by id. Similarly for class document.getElementsByClassName(). There are other selectors in JS. querySelector() and querySelectorAll() etc.

What is Event delegation in the context of the DOM, and why is it useful?

Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate attributes : "setAttribute()" or "getAttribute()" method.

To manipulate styles : "element.style" is used. There is also other ways.
